# Parking Lot

This file is used to quickly capture thoughts, ideas, concerns, risks, features, implementation notes, questions, or reminders without interrupting the current workflow.

Items in this file do not need to be classified immediately.

## Workflow

1. Capture the thought quickly.
2. Continue the current task.
3. Review the parking lot periodically.
4. Move each item to the correct place:
   - Product Definition and Requirements
   - Decision Log
   - Open Questions
   - Roadmap
   - Archive / Discard

## Review Status Values

- New
- Reviewed
- Moved
- Discarded
